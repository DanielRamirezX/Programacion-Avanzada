El objetivo no es aprender a programar.
Es aprender a pensar como programador.

Al finalizar el curso el alumno será capaz de desarrollar aplicaciones orientadas a objetos usando Python como lenguaje base, con perspectiva comparada en Java, JavaScript y Go. Clases, arreglos, herencia, excepciones y conexión a bases de datos — con un proyecto real construido sprint a sprint. La IA generativa es herramienta y contenido del curso: prompt engineering, agentes y MCPs.
